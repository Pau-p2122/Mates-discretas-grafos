El presente proyecto, tiene como objetivo aplicar teorias de Grafos Matematicos para determinar rutas mas convenientes desde un punto de origen, hasta un punto de destino.
Se utiliza codigo en Phyton para desarrollar una aplicacion donde se realizara el calculo de rutas directas, 
tambien se determina una seleccion de 15 ciudades y sus correspondientes conexiones entre si, estas determinadas en rutas terrestres reales, investigadas en aplicacion de geolocalizacion.
el algoritmo de calculo utilizado corresponde a el algoritmo de Dijkstra, utilizando este de forma manual en el codigo de Phyton.
